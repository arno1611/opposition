===============================================================================
                           README - Site Oppose
===============================================================================

===============================================================================

Ce site web a été développé dans le but de fournir des informations et des services
liés à l'opposition d'une ordonnance d'injonction de payer. Il utilise une
combinaison de HTML, CSS (Bootstrap), JavaScript (jQuery), et PHP pour offrir une
expérience utilisateur optimisée et fonctionnelle.

Contenu du site:
-----------------

1. Structure du dossier:
   - assets/           : Contient les ressources telles que les images et les scripts JavaScript.
   - pages/            : Pages principales du site comme l'accueil, à propos, services, etc.
   - forms/         : Fichiers PHP inclus pour les envois de emails contact form.

2. Technologies utilisées:
   - HTML5 & CSS3 avec Bootstrap pour le style et la mise en page responsives.
   - JavaScript avec jQuery pour les fonctionnalités interactives.
   - PHP pour la gestion des formulaires de contact.

3. Fonctionnalités principales:
   - Navigation claire et intuitive avec un menu de navigation responsive.
   - Section FAQ avec des questions et réponses expansibles.
   - Formulaire de contact pour les utilisateurs désirant des informations supplémentaires.
   - Section tarifs détaillant les niveaux de service proposés avec des exemples de coûts.

4. Personnalisation:
   - Les couleurs, polices et mises en page ont été conçues pour être modernes et professionnelles,
     correspondant à l'image de marque d'Oppose.
